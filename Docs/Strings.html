Strings.h

Implementation 
+---------------------------------------------------------------+
Provides generic functions for handling strings and converting
other types to strings. 
Also contains the mlstring (multi-line string) struct, which is
used for operating on large strings.

Also includes:
DataStructures.h


constants:

string ANSIESC:
    ansi escape sequence ("\33[")
enum ansiColor:
    {BLACK,RED,GREEN,YELLOW,BLUE,MAGENTA,CYAN,WHITE,DEFAULT}
    PURPLE=MAGENTA
string ANSITXT[9] and string ANSIBKG[9]:
    Array of color codes for corresponding colors, take
    ansiColor as index


functions:

size_t psize(string)
    returns number of printible characters in string
    Useful for formatting output when multiple lines should 
    be equally long, but you don't want to count newlines or 
    ansi escape codes like colors.

string pad(string&/char*, int Width, char Fill=' ', string align="left")
    adds 'Fill' to the string until it is 'Width' 
    characters long

mlstring pad(mlstring&, int Width=-1, char Fill=' ', string align="left")
    pads each line of mlstring, otherwise synonymous with above.

mlstring align(mlstring a,mlstring b,
                      sep="",style="horizontal")
    Aligns two mlstrings in a certain style
     - horizontal: placed next to each other like 'a[i]+sep+b[i]'
     - vertical:   b is appended to a

string setColor/setHighlight (string, ansiColor)
    returns string with corresponding color codes applied

string toLower(string&/char*)
    converts all uppercase letters to lowercase

mlstring addBorder(mlstring,title="")
    Adds a border to mlstring. Example:
    +----+    +-----+
    |text| or |title|
    +----+    +---------+
              |some text|
              +---------+

string addBorder(string/char*, title="")
    Same as above, but only for a single line of text.

string niceDeci(double)
    truncates double to 2 decimal places, then rounds .99 up
    and removes .00's.

constructor:

mlstring(text="")


Methods:

string& at(n)
    Return reference to line n

string& operator[n]
    Return reference to line n
    Synonymous with at(n)
    Called like "VARNAME[n]"

mlstring& operator+= string/char
    Appends text to the string, handles newlines
    returns a reference to itself so you can chain 
    operators like "VARNAME += a += b"

size_t size()
    returns number of lines

size_t maxWidth()
    returns the largest width among all lines

string toString(seperator="\n")
    returns a formatted string of all lines

bool in(string&)
    used for looping through lines in a for loop
    returns false when end of lines is reached
    example: for(string i;VARNAME.in(i);){...}
