BinaryTree.h

Implementation 
+---------------------------------------------------------------+
Provides a binary search tree struct which can store and quickly
search elements. A binary tree is very quick at checking if it 
contains a certain value, making it useful for many applications.
Notably any duplicate elements are discarded.

Also includes:
Vectors.h


struct BinaryTree:


constructor:

BinaryTree<typename T>(vector<T> elements)
BinaryTree<typename T>(T value)
BinaryTree<typename T>()


Methods:

void push(T object)
    adds an object to the tree

bool has(T object)
    returns true if the tree contains object

vector<T> collapse()
    returns a sorted vector containing all of the trees elements.

void resize()
    reorganizes the tree to more evenly distribute the elements,
    this helps keep the height low, in turn making lookup times 
    more efficient. It is up to the user to determine when to
    resize the tree, usually based on calls to height()

size_t height()
    return the length of the longest branch in the tree.

string toString()
    Almost only useful for debugging, outputs a string depiction
    of the tree. There are multiple issues with this, as it only
    outputs the first character of each element, and only works
    when there is a valid call to to_string(T val)



struct TreeNode:

This file also contains a TreeNode struct used by the 
binary Tree, but for most use cases this isn't necessary 
to interact with or know about.

constructor:

TreeNode<typename T>(T obj)


Methods:

bool attach(T object)
    Appropriately attaches the object, or recursively passes
    it down to this node's children. 
    Returns false if object is a duplicate element in the tree.

bool has(T object)
    Returns true if this node or it's children contain
    object.

vector<T> collapse()
    returns a sorted vector of this node and it's 
    children's values
