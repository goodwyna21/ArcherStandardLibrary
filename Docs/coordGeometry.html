coordGeometry.h

Implementation 
+---------------------------------------------------------------+
Provides 'Vec' (vector) and 'coordPlane' classes for use in
coordinate geometry.

also includes:
<vector>
<cmath>
<initializer_list>


Data types:

uint = unsigned int


struct Vec:


constructors:

(by default, values are initialized to 0)
Vec(dimension=0)
Vec(dimension=0, double values[])
Vec(vector<double>& values)
Vec(Vec& v)
Vec(initalizer_list<double>& values)
  - ex. "Vec v = {a,b...};"


members:

uint Dimension
vector<double> coords


methods:

Vec * clonePntr()
    returns a pointer to a copy of this object

Vec clone()
    returns a copy of this object

string toString(string space="")
    returns a string representation of this

operator vector<double>()
    implicit conversion to vector of coordinates

double& operator[](uint index)
    return a reference to the coordinate at index

double dist(Vec& v)
    returns distance to v
double distSquared(Vec& v)
    returns distance to v squared
    (more accurate/efficient than dist)

double norm()
    returns this vector's norm (length)
double normSquared()
    returns this vector's norm (length) squared
    (more accurate/efficient than norm)

double angleTo(Vec& v)
    returns the angle formed by this vector and v (radians)

double dot(Vec& v)
    returns the dot product of this vector and v


vector operations:

('a' and 'b' are Vec objects with equal dimension)
(a+=b) or (a+b)          :  vector addition
(a-=b) or (a-b)          :  vector subtraction
(a*=n) or (a*n) or (n*a) :  scalar multiplication



struct coordPlane:


constructors:

coordPlane(symbol="")
coordPlane(Vec origin, Vec ĵ, Vec k̂, symbol="")
    origin: 3d position of 〈0,0〉
    ĵ:      horizontal unit vector (from origin to 〈1,0〉)
    k̂:      vertical unit vector (from origin to 〈0,1〉)
    symbol: label


members:

Vec j, k, origin
string sym


methods:

Vec toRealSpace(Vec v)
    returns the 3d position of 2d vector v on this plane
Vec operator()(uint x, uint y)
    returns the 3d position of 〈x,y〉 on this plane
